Introduction

Analysis of Tactile Map Reading (ATMR) is a project to produce instrumentation to help assistive cartography researchers understand the tactile map-reading behavior of visually-impaired users. The visual finger-tracking group developed software to track finger motion by extracting finger tip positions from video. Tracks of motion are produced in a common XML format agreed among the tablet, visual, and signal processing teams.

Key features of the video processing component include:

  • Load Videos: Loads uncompressed AVI videos and makes stacks of frames
  • User Interface: Designate the finger tip associated with each motion track.
  • Position extraction : Finds the centroids of the fingers with specific color patches.
  • Output XML File: Outputs the extracted information with time stamps and finger IDs.

For developers please visit the ATMR Visual project space.
