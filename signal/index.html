Introduction

Analysis of Tactile Map Reading (ATMR) is a project to produce instrumentation to help assistive cartography researchers understand the tactile map-reading behavior of visually-impaired users. The signal team constructed a first prototype of software to combine, analyze, and display motion tracks produced by the tablet and visual components. Video is loaded when a test subject is selected and upon pressing play, graph(s) are dynamically drawn that represent certain parts of the collected test data. Display can include a representation of the tactile map presented to the user as well as a graph that encodes the path fingers travel and the duration a finger remained at each position. In addition, there are areas to type notes and descriptions on the chosen test case, that can be saved for later access.

For developers please visit the ATMR Signal project space.
